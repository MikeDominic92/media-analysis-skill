## Overview
Unified media analysis skill for automatic ticket file processing in BMAD-EDI workflow.
Combines Google Gemini 2.5 Pro with PaddleOCR for multimodal analysis and text extraction.

## Features Implemented

### Phase 0: Pre-Investigation Analysis
- Automatic file detection (PDF, images, audio, video)
- Gemini 2.5 Pro multimodal analysis
- PaddleOCR text extraction with preprocessing
- Hybrid mode combining both analyzers
- EDI metadata extraction (ticket ID, company, trading partner, transaction type)
- Confidence scoring (0.0-1.0 scale)
- Standardized filename generation
- Automatic file organization (incoming → processing)
